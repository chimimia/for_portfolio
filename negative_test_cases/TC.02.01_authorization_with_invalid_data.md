## Название тест-кейса: Авторизация на сайте с использованием некорректных данных

**Описание:** Тест проверяет возможность авторизации пользователя при введении пароля с опечаткой

**Шаги теста:**
1. Открыть веб-сайт: https://www.saucedemo.com/;
2. В поле "Username" ввести значение "standard_user";
3. В поле "Password" ввести значение "secrt_sauce";
4. Нажать на кнопку "Login"

**Ожидаемый результат:**
После нажатия на кнопку "Login" появляется сообщение об ошибке: "Epic sadface: Username and password do not match any user in this service"
